# Analyses for the Belize GBIF Data Impact Brief

This repository holds the code to accompany the UB-ERI data brief 'Belize’s Contributions to Global Biodiversity Data,' including data access, analysis, and visualization.

To run this project, simply run the script `project_run.r`.


## Respository Structure

- `> data`- Folder to store data pulled through APIs 
- `> outputs` - Folder to store outputs generated through the script
- `> shapefiles` - Folder containing Belize shapefiles
- `data_analysis.r` - Script to analyze data after it is downloaded; run automatically in `project_run.r`
- `data_download.r` - Script to download data; run automatically in `project_run.r`
- `LICENSE.txt` - Creative commons license for the project
- `packages_attach.r`- Script to install and attach project packages; run automatically in `project_run.r`
- `project_run.r` - Script to run the project. **Key script!**
- `README.md` - This file
- `theme_define.r` - Script to define color palette and ggplot theme for visualizations


## Brief

Maximizing impact is often the desired destiny of a dataset, once collected. Ideally, our biodiversity data will not only achieve its intended function within the bounds of a monitoring program or project, but additionally create impact outside that scope, contributing to wider initiatives both regional and global. However, for this to occur, the data must be accessible to a  wider audience, findable, and in a usable format. Here, we explore how the online Global Biodiversity Information Facility (GBIF) can serve that purpose for biodiversity data, and how Belize’s existing GBIF contributions are already generating impact on the global academic stage.
